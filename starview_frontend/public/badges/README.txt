STARVIEW BADGE ICON FILES
================================================================================

This directory contains badge icon files (.png format) for the Starview
badge/achievement system. Each badge requires a corresponding icon file.

REQUIREMENTS:
- Format: PNG
- Naming: Must match the filenames listed below exactly (lowercase, hyphenated)
- Recommended size: 256x256px or 512x512px
- Transparent background preferred

================================================================================
NOTES:
================================================================================

- These icons will be displayed in user profiles, badge lists, and achievements
- Icons are served from /badges/[filename] in the frontend
- Missing icons will result in broken images in the UI
- For best results, use consistent visual style across all badge icons
- Consider color coding by category (e.g., gold for high-tier badges)

Badge categories and their typical color schemes:
  - EXPLORATION: Blue/teal (journey/discovery theme)
  - CONTRIBUTION: Green/emerald (growth/adding value)
  - QUALITY: Gold/yellow (excellence)
  - REVIEW: Purple/violet (wisdom/insight)
  - COMMUNITY: Orange/coral (connection/warmth)
  - SPECIAL: Pink/magenta (unique achievements)
  - TENURE: Purple (prestige/longevity)
